# Weights

This repo intentionally does not vendor large checkpoints.

## Backbone pretraining weights

Backbone pretraining weights are handled by `timm` when `pretrained=True`.

## Paper checkpoints

Trained SARDet-100K checkpoints (and other large artifacts) should be published outside git
(e.g., GitHub Releases for this repo).

See `docs/ARTIFACTS_AND_RELEASES.md` for the repo policy and recommended bundle format.
